Pi Bot

Pi Bot is an exciting new robot being developed for the Raspberry Pi computer. It is aimed at digital makers wanting to learn and have fun with technology.

Open Source

The Pi Bot's design and software is open source - this means that you are free to tinker with the Pi Bot and change it in what ever way you want.

Made for Makers

Pi Bot is made from a Do-It-Yourself kit. It's fun and easy to put together and you'll learn something in the process.

For the Raspberry Pi

The Pi Bot is designed for the Raspberry Pi mini-computer. Find out more about the Raspberry Pi here.
