Want to build a robot? The PiBot kit is easy to build and comes with everything you need to get started

PiBot can be programmed to do anything you can think of, either by writing code yourself or using a child-friendly puzzle-block interface

PiBot comes pre-programmed with a number of functions, from reacting to movement to singing a song. And you can add more!
